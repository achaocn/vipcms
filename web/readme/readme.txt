一、环境需求
PHP：5.4  5.5
MYWSQL  5.5  5.6
空间>=500M

二、操作
文件解压后放到网站根目录 保证/data/有读写权限
打开您的域名 回跳到配置数据库页面，按照提示会配置好数据库，提示成功后，
此时您可以进入网站后台 http://您的域名/admin.php 设置您的授权码，授权码填写完成后清除缓存网站就可以打开了
后台默认用户名=admin  密码=admin

三、注意事项
定期清理缓存 以免缓存过多导致网站空间被占满
服务器时间必须是北京时间，不能有误差

四、如果数据库导入不成功，可以用数据库工具导入/readme/vipcms_db.sql文件，把/readme/下的install.lock文件复制到/data/下;修改数据库配置文件 文件目录/data/config/db.php

五、所有配置成功后为了安全请删除/install.php文件
